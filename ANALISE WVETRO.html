O Que NÃO Pode Ser Implementado

Limitação Descrição Alternativa Referência
Cálculos Estruturais Complexos Análises de resistência avançadas, simulações de vento/cargas sísmicas Usar software especializado complementar Nota Técnica WVETRO 12.3
Customizações Extremas Perfis totalmente personalizados, componentes não cadastrados Solicitar inclusão ao administrador do sistema Política de Customização WVETRO
Integrações com Software Externos Integração direta com CAD, ferramentas de análise estrutural Usar exportação/importação manual via formatos compatíveis Guia de Interoperabilidade 2.4
Automatização Completa Substituição do conhecimento técnico do projetista Manter validação humana nos pontos críticos Manual de Boas Práticas WVETRO
Variações Regionais Automáticas Adaptação automática a diferentes normas regionais Criar modelos separados para cada região Guia de Conformidade Regional
